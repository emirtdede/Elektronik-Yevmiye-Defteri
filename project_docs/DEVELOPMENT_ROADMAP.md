# Geliştirme Yol Haritası

## V1 — Temel İskelet ve İşçi Yönetimi (Şu an)
- [ ] Electron, React ve SQLite kurulumlarının yapılıp projenin ayağa kaldırılması.
- [ ] Veritabanı tablolarının (migration) oluşturulması.
- [ ] İşçi Ekleme, Listeleme, Düzenleme (CRUD) modüllerinin tamamlanması.
- [ ] İşçi detay/profil ekranının temel arayüzünün yapılması.

## V2 — Puantaj ve Finansal Hareketler
- [ ] Puantaj giriş modülü (Tam, yarım gün vb. işaretleme).
- [ ] İşçiye göre otomatik hak ediş hesaplama mantığının kurulması.
- [ ] Avans, Kesinti ve Ödeme giriş ekranlarının tamamlanması.
- [ ] İşçi bakiyesinin (Alacak - Borç = Kalan) anlık olarak profiline yansıtılması.

## V3 — Kasa ve Excel İçe Aktarım
- [ ] Merkez Kasa modülü (İşçiye yapılan ödemelerin kasadan düşülmesi).
- [ ] Dosya yükleme modülü (Excel/CSV seçme ve önizleme).
- [ ] Excel sütunlarını veritabanı alanlarıyla eşleştirip içeri aktarma (Import) mekanizması.

## V4 — Raporlama ve Çıktı Üretimi
- [ ] İşçi bazlı "Aylık Hesap Ekstresi" oluşturulması.
- [ ] Ekstrenin PDF formatında dışa aktarılması (Export) ve yazdırılabilir hale getirilmesi.
- [ ] Şirket genel işçilik maliyeti ve avans toplamlarını gösteren Dashboard tasarımı.
